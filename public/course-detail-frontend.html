Deskripsi Kursus

Kursus ini dirancang untuk pemula yang ingin mempelajari dasar-dasar pengembangan web. Anda akan belajar HTML, CSS, JavaScript, dan framework populer seperti React dan Tailwind CSS. Di akhir kursus, Anda akan mampu membangun website responsif dan dinamis.

Materi yang Dipelajari

  • Pengenalan HTML, CSS, dan JavaScript
  • Responsive Design dengan Tailwind CSS
  • Dasar-dasar React
  • Deploy Website ke Internet

Persyaratan

  • Komputer dengan koneksi internet
  • Semangat belajar dan berlatih

Jadwal & Durasi

Kursus berlangsung selama 8 minggu, setiap Sabtu dan Minggu pukul 09.00 - 12.00 WIB.

Curriculum

  1. Week 1-2: Dasar HTML & CSS
  2. Week 3: JavaScript Fundamental
  3. Week 4: Responsive Design dengan Tailwind CSS
  4. Week 5: Pengenalan React
  5. Week 6: Project Mini Website
  6. Week 7: Deployment & Hosting
  7. Week 8: Review & Final Project
